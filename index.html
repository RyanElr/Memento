LA LIGNE DE COMMANDE

Gestion de documents

Pour créer un dossier je dois utiliser cette commande : mkdir

Pour créer un fichier je dois utiliser cette commande : touch

Pour créer un dossier et un sous dossier je dois utiliser cette commande : mkdir -p

Pour déplacer un fichier ou un dossier je dois utiliser cette commande : mv

Pour se déplacer vers un dossier je dois utiliser cette commande : cd

Pour lister les fichiers et les sous-dossiers je dois utiliser la commande : ls

Pour obternir plus d'informations (version longue) je dois utiliser la commande : ls -l

Pour aussi afficher les fichiers/dossiers cachés je dois utiliser la commande : ls -a

Pour mélanger les deux options je dois utiliser la commande : ls -Al

Pour afficher le chemin du dossier je dois utiliser la commande : pwd

Pour faire une copie d'un fichier/dossier je dois utiliser la commande : cp

Pour passer en super utilisateur je dois utiliser la commande : Sudo -s

Installation et mise à jour des paquets

Pour installer un paquet je dois utiliser la commande : sudo apt install

Pour forcer une installation d'un paquet je dois utiliser la commande : sudo dpkg -i

Pour rechercher un paquet je dois utiliser la commande : apt search

Pour recherche une mise à jour je dois utiliser la commande : sudo apt update

Pour effectuer une mise à jour je dois utiliser la commande : sudo apt upgrade

Pour réparer des paquets défectueux je dois utiliser la commande : sudo apt --fix-broken install
