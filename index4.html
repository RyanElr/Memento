Git

Pour créer son git je dois utiliser la commande: "git --global user.name" "git config --global user.email"

Pour ajouter un fichier à l'index git j'utilise la commande: Git init

Pour gérer un repository j'utilise la commance : git add " plus nom du fichier"

Pour afficher la liste de tout les commit j'utilise la commande: git log.

Pour insérer un message dans le git j'utilise la commande: git commit -m "plus le nom du fichier".

Pour pousser le git sur le site github j'utilise la commande: git push origin master.

Pour libérer le repository j'utilise la commande : git remote add origin + le lien du repository
